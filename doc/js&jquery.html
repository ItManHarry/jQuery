jQuery Selector
Selector Class Selector Remark
Attribute Selector $("a") All "a" elements
$("a[title]") All "a" elements which has "title" attribute
$("a[href=test.html]") The "a" element whose "href" attribute's value is "test.html"
$("a[href^=http]") All "a" elements which "href" begin with "http"
$("a[href$=html]") All "a" elements which "href" end with "html"
$("a[href*=acre]") All "a" elements which "href" contains "acre"
Container Selector * All elements
E All "E" elements
E F All "F" elements, and "E" is "F" 's parent node,contains all childNodes
E > F All "F" elements, and "E" is "F" 's parent node,don't contain any childNodes
E + F All "F" elements, and "F" following the "E" element
E ~ F All "F" elements, and before "F" there is one "E" element
E:has(F) All "E" elements, whick contain "F" elements
E.C All "E" elements, which class name is "C"
E#I All "E" elements, which id is "I"
E[A] All "E" elements, which has "A" attribute
E[A=V] All "E" elements, which "A" attribute is "V"
E[A^=V] All "E" elements, which "A" attribute begins with "V"
E[A$=V] All "E" elements, which "A" attribute ends with "V"
E[A*=V] All "E" elements, which "A" attribute contains "V"
Position Selector :first The first element, Example:($("p:first"):the first p element, $("div p:first"))the first p element, and the p is the div's child element
:last The last element, Example:($("p:last"):the last p element, $("div p:last"))the last p element, and the p is the div's child element
:first-child The first element, Example:($("p:first-child"):the first p element, and the p is its parent element's first child element
:last-child The last element, Example:($("p:last-child"):the last p element, and the p is its parent element's last child element
:only-child The only element, Example:($("p:only-child"):the only p element, means its parent has only one child element
:nth-child(N) The Nth element, index begins with 1
:nth-child(odd|even) The odd or even elements
:nth-child(nX + Y) The (nX + Y)th element, n begin with 0
:odd or :even The odd or even elements for the whole page
:eq(n) The nth element , begin with 0
:gt(n) All elements after the nth element
:lt(n) All elements before the nth element

Manage The Selected Result
Method Remark
size() The quantity of selected elements,Example:$("div").size()
get(index) Get the element from the selected elements.Example:$("div").get(2) or $("div")[2]
add() Add more element to the selected result.Example:$("img[alt]").add("img[title]") equals $("img[alt],img[title]")
not() Remove some elements from the selected result.Example:$("img[title]").not("[title*=osaac]")
filter() Filter the selected result, it can be used in two ways.Example:(1).Using like "not()" method:$("img[title]").filter("title*=osaac")),$("div").addClass("cs1").filter("[class*=middle]").addClass("cs2");(2)Use function as parameter:$("div").addClass("cs1").filter(funtion(index){ return index == 1 || $(this).attr("id") == "fourth" }).addClass("cs2")
find() Get elements from the selected result.Example:$("p").find("span") --- Get the spans which in the p tag
is() Whether one element has another element .Example:$("p").is("img") --- Whether p tag has img tag
end() Operate the last object.Example:$("p").find("span").addClass("cs1").end().addClass("cs2"),Result : set the span's class cs1, p's class cs2
andSelf() Opetate all elements.Example:$("div").find("p").addClass("cs1").andSelf().addClass("cs2").Result:p's class cs1 and cs2 , div's class cs2
each() Iterator each element .Example:$("div").each(function(){...})
attr(name) Get attribute value .Example:$("div").attr("id")
attr(name,value/fn) Set attribute value .Example:$("div").attr("id","002"),$("div").attr("name",function(){...})
removeAttr(name) Set attribute value the the default value.Example:$("div").removeAttr("disabled")
addClass(classList) Add CSS style(one or more styles, split with space).Example:$("div").addClass(css1 css2 css3 ...)
removeClass(classList) Remove CSS style(one or more styles, split with space).Example:$("div").removeClass(css1 css2 css3 ...)
toggleClass(class) Dynamic change CSS style.Example:$("div").toggleClass(css)
css(name) Get one CSS property value .Example:$("div").css("width")
css(name,value) Set one CSS property value .Example:$("div").css("width", "200")
css(properties) Set one CSS property value .Example:$("div").css({name1:value1,name2:value2,...})
html() Get html value .Example:$("div").html()
text() Get text value .Example:$("div").text()
html(htmlValue) Set html value .Example:$("div").html("<p>html value</p>")
text(textValue) Set text value .Example:$("div").text("text value")
append(node) Add new child node .Example:$("div").append($("<p>new node</p>"))
appendTo(target) Add node to the target node .Example:$("img:eq(0)").appendTo($("div:eq(0)"))
prepend(target) / prependTo(target) Add node to the target node as the target node's first child node .Example:$("img:eq(0)").prepend($("div:eq(0)"))
before(target) / insertBefore(target) Add node before the target node(not as child node,the target is the first node) .Example:$("img:eq(0)").before($("div:eq(0)"))
after(target) / insertAfter(target) Add node after the target node(not as child node,target is the first node) .Example:$("img:eq(0)").after($("div:eq(0)"))
remove() Remove the node .Example:$("img:eq(0)").remove()
empty() Remove all the child nodes .Example:$("img:eq(0)").empty()
clone() Clone the node.Example:$("img:eq(0)").clone().appendTo($("p:eq(0)"))
val() Get form element's value .Example:$("#uname").val()
val(value) Set form element's value.Example:$("#uname").val("Harry")
jQuerty Event
Method Remark
bind(eventType,[data],Listener) Bind action listener to the event type.Example:$("button:eq(0)").bind("click", function(){...})
one(eventType, listener) Bind action listener to the event type, but it only works once.Example:$("button:eq(0)").one("click", function(){...})
unbind([eventType],[fnName]) Unbind action listener of the event type.Example:(1),$("button:eq(0)").unbind();(2),$("button:eq(0)").unbind("click");(3),$("button:eq(0)").unbind("click", fnName)
trigger(eventType) Trigger action listener of the event type.Example:$("button:eq(0)").trigger("click")
toggle(fn1, fn2) Toggle action listener,only for click action .Example:$("button:eq(0)").toggle(function(){...},function(){...})
hover(fn1, fn2) Toggle action listener,for mouse over action .Example:$("button:eq(0)").hover(function(){...},function(){...})
